# HR-Analytics-Dashboard
A comprehensive HR Analytics Dashboard designed to help organizations optimize workforce management through actionable insights using data visualization. This Dashboard used to analayze employee data such as Attrition Rate to support strategic HR decision-making.

This project presents an interactive HR Analytics Dashboard built using Power BI to analyze employee data and uncover actionable insights related to workforce management, attrition, and employee demographics.
The dashboard is designed to help HR professionals and decision-makers understand key patterns in employee behavior and improve strategic planning.

Objective
The main goal of this project is to:
Analyze employee data effectively
Identify key factors influencing attrition
Support data-driven HR decisions

Key Features
1-Workforce Overview
Total Employees: 1.47K
Active Employees: 1K
Attrition Count: 237
Attrition Rate: 16.12%
Average Age: 37

2-Attrition Analysis
Department-wise attrition (R&D, Sales, HR)
Education field-wise attrition (Life Sciences, Medical, Marketing, etc.)
Gender-based attrition trends across different age groups

3-Demographic Insights
Employee distribution by age group (Under 25 to Above 55)
Gender comparison across age categories

4-Job Satisfaction Analysis
Role-based satisfaction ratings (1–4 scale)
Identification of roles with high/low satisfaction levels

5-Dynamic Filtering
Education-level filters (Associate, Bachelor’s, Master’s, Doctoral, High School)
Interactive visuals for deeper exploration

Tools & Technologies
Power BI – Data visualization and dashboard creation
Data Cleaning & Transformation – Power Query
DAX (Data Analysis Expressions) – KPI calculations and measures

Enhance employee retention strategies
